# Дневник тренировок

Приложение для отслеживания тренировок и прогресса.

## Функционал

- ✅ Три группы тренировок: Грудь, Спина, Ноги
- ✅ Добавление собственных упражнений (бицепс, трицепс, предплечья, трапеции, плечи)
- ✅ Индивидуальные данные для каждого подхода (вес, повторения)
- ✅ Сохранение состояния между сессиями
- ✅ Статистика и прогресс по упражнениям
- ✅ Экспорт в CSV
- ✅ Мобильная версия

## Разработка

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Развертывание на Netlify

### Через сайт (самый простой способ)

1. Откройте https://app.netlify.com/
2. Нажмите "Add new site" → "Upload files"
3. Перетащите папку `workout-tracker` в окно загрузки
4. Нажмите "Deploy site"

Netlify автоматически:
- Установит зависимости из `package.json`
- Соберёт проект
- Опубликует сайт

### Через Netlify CLI

```bash
npm install -g netlify-cli
netlify deploy --prod
```

### Через Git

1. Создайте репозиторий на GitHub
2. Подключите его к Netlify
3. Деплой будет происходить автоматически после каждого push
